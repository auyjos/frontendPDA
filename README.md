
## Install

```
npm i @richardev/vue3-boilerplate
```

or

```
git clone https://github.com/richardevcom/vue3-boilerplate.git
```

then

```
cd vue3-boilerplate
npm install
```

## Run / Develop

```

npm run dev

```

## Build

```

npm run build

```

## Preview

```

npm run preview

```

## Lint

```

npm run lint

```
